CSS Transform a Skew

O que é skew?

A função skewno CSS é usada para inclinar ou distorcer um elemento ao longo de seus eixos X e Y. A inclinação é especificada em graus deg.

Tipos de skew

  • skewX(valor):Inclina o elemento ao longo do eixo X.
  • skewY(valor):Inclina o elemento ao longo do eixo Y.
  • skew(x, y):Inclina o elemento ao longo do eixo x e y.
